# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: treat people with respect, no harassment, no discrimination, no name-calling. Disagreement on technical merits is welcome; making it personal isn't.

## Enforcement

Conduct issues go to **conduct@zibby.app**. We'll review within 5 business days and respond with what action (if any) we'll take. Outcomes can include a private warning, a public warning, temporary loss of contribution privileges, or a permanent ban — chosen proportional to the offense and prior history.

We won't publish complainants' names without consent. We'll publish a summary of significant actions taken so the community knows enforcement is happening.

## Scope

Applies anywhere this project's name shows up — GitHub issues / PRs, the project's Discord / Slack if/when those exist, conference talks, social media when speaking as a contributor.
